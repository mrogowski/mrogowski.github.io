Biometrics

Jun 29, 2021

Starting in 2008, as a second year Computer Science B.Sc. student at the Bialystok University of Technology, I started working on keystroke dynamics with Prof. Khalid Saeed. Keystroke dynamics is one of biometric methods that can be used to authenticate or identify users. The method falls under behavioral features and is based on how a user performs certain tasks - in this case - how user types on a keyboard. The typing style was shown to differentiate well between users. The idea first came as early as 1975 when it was mentioned by R. J. Spillane and has since then been modified and improved on multiple occasions with the results reported as high as 99-100% correct classification. In general, in almost all the prior work on keystroke dynamics the user is asked to type a particular password multiple times. The method can be used for both authentication (strenghtening the password or continuously verifying the user’s identity) or identification (determining the user based on his typing style).

My work started by analyzing the impact of the database quality on the method’s efficiency and continued with the implementation of a JavaScript-based website to collect experimental data. My biggest contributions, however, were in the area of user authentication on mobile devices. Back in 2012, when smartphones started rapidly gaining popularity, I researched methods of authentication on mobile devices and proposed keystroke dynamics as one of the solutions. I then implemented a custom Android keyboard that was collected extra information from the touchscreen and the gyroscope as users were typing. The resulting ensemble of data showed promising results.
